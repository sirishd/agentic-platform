# Agentic AI Platform Lifecycle

An interactive visualization of how AI agents fit into the enterprise software development lifecycle.

## Deploy to Vercel (fastest)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) and click "New Project"
3. Import your GitHub repo
4. Vercel auto-detects Vite — just click "Deploy"
5. Your site is live in ~30 seconds

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

Output goes to `dist/` — can be deployed anywhere that serves static files.

## LinkedIn Open Graph

The `index.html` includes OG meta tags for a rich LinkedIn preview. To complete the setup:

1. Take a screenshot of the visualization
2. Save it as `public/og-image.png` (1200x630px recommended)
3. Redeploy

LinkedIn will show the image as a rich card when you share the link.
